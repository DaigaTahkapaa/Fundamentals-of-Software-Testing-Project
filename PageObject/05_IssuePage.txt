### **Issue functionality Testsuite (TS-5)**

*** Settings ***

Documentation	Testing Issue CRUD (Create, Read, Update, Delete) within a seed project. Focuses on creating issues, viewing details, editing title/description, and deleting issues.
				Scope limitations:
					- Issue type only (Incidents and Tasks excluded)
					- No board/epic/label automation
					- No attachments
					- Single private project seed used for all cases
					- Tests navigate via project-level Issues page; dashboard/My Work global issue views excluded.

Pre Condition	Internet connection is active
				User has a valid registered account
				Browser is open in incognito/private mode
				Navigate to LoginUrl
				Sign in with valid credentials

Post Condition	Delete any issues created during testing; delete the seed project
				Ensure user is logged out
				Close incognito browser window

Tags			Issue, CRUD, TS-5-Issue
